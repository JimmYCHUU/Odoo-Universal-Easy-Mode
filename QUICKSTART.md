# Quick Start Guide - Universal Easy Mode

## 🚀 Quick Installation (5 minutes)

### Step 1: Install the Module
1. Copy the `universal_easy_mode` folder to your Odoo addons directory
2. Restart Odoo server
3. Go to **Apps** → Update Apps List → Search "Universal Easy Mode" → Install

### Step 2: Create Demo Configuration (Recommended)
1. Go to **Easy Mode** → **Tools** → **Create Demo Configuration**
2. This automatically configures the Contacts module for testing

### Step 3: Test Easy Mode
1. Go to **Contacts** (or any configured module)
2. Look for the 📱 **"Easy Mode"** button in the top-right systray
3. Click to toggle between Easy Mode and Full Mode
4. Notice the simplified form with only basic fields

## ⚙️ Full Configuration (10 minutes)

### Step 1: Configure Modules
1. **Easy Mode** → **Settings** → **Easy Mode Configuration**
2. Create new configuration
3. **Module Selection** tab: Choose modules (e.g., Contacts, Sales)
4. Click **"1. Refresh Models"**

### Step 2: Configure Fields
1. **Model Configuration** tab: Enable desired models
2. Click **"Configure Fields"** for each model
3. Mark fields as Basic (📱) or Advanced (🔧)
4. Use **"Auto Classify"** for quick setup

### Step 3: Generate Views
1. Click **"2. Generate Easy Views"**
2. Views are now ready for use!

## ✅ Validation
Run **Easy Mode** → **Tools** → **Validate Installation** to check everything is working.

## 🎯 Usage
- Users see 📱 **Easy Mode** toggle in systray
- Green button = Switch to Easy Mode (simplified)
- Yellow button = Switch to Full Mode (complete)
